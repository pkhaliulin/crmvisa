Engine: PHPUnit + SQLite in-memory (~5 секунд). CI: GitHub Actions.

Ключевое ограничение: SQLite vs PostgreSQL
Все 167 тестов выполняются на SQLite in-memory. PostgreSQL RLS — главный защитный механизм — не тестируется автоматически. Необходим отдельный integration test suite на реальном PostgreSQL.
